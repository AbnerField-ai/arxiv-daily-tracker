FlashRT: Agent Harness for Guiding Agents to Deploy Real-Time Multimodal Applications

Krish Agarwal, Zhuoming Chen, Yanyuan Qin et al.

问题：实时多模态应用（如语音代理和交互式视频生成）的高效部署需要针对特定应用进行放置、流式和模型内并行等决策，但现有系统无法灵活适应新应用，导致需要手工优化。
方法：提出FlashRT，一种代理框架，通过链式编程范式引导编码代理将简单的参考实现转换为优化的多GPU部署：代理将参考代码转换为中间表示（IR）以捕获数据依赖和持久状态，通过顺序解释器验证IR，进行静态分析识别候选变换，然后在测量门控优化循环中迭代实现、验证和基准测试每个候选方案。
结论：在NVIDIA B200 GPU上实现高达70倍延迟降低和2.8倍吞吐量提升；在AMD MI355X GPU上实现3.6倍吞吐量提升，且对Qwen3-Omni文本到音频推理的响应延迟比专家vLLM-Omni实现降低65%。
创新点：首次提出利用编码代理通过多阶段变换和测量门控优化循环自动优化实时多模态应用的部署，无需手工专家知识。
意义：为AI系统部署自动化提供了新范式，尤其对多模态大模型和世界模型等新兴应用的实用化具有重要推动作用。

SAGE: Subgoal-Conditioned Action Generation for Latent World Model Planning

Letian Cheng, Qi Zhang, Yisen Wang

问题：解决潜在世界模型在长时域规划中因固定候选动作预算无法有效探索指数级增长的动作空间，导致性能受限的问题。
方法：提出一种基于子目标条件的规划器，通过目标条件生成器预测可达到的潜在子目标，并以此作为先验条件生成候选动作序列，利用不同时域长度的子目标平衡局部控制与长期进展。
结论：在PushT和OGBench Cube任务上，长时域规划成功率显著提升（如目标偏移150时，PushT从12.7%升至64.7%，OGBench Cube从26.7%升至67.3%），同时保持短时域性能。
创新点：将潜在子目标分解与先验条件动作生成相结合，替代随机动作初始化，实现结构化规划引导。
意义：为潜在世界模型在长时域复杂任务中的规划提供高效方法，推动具身智能和机器人控制领域发展。

The Many Senses of Visual Similarity: A Text-Prompted Image Perceptual Metric

Sheng-Yu Wang, Yotam Nitzan, Aaron Hertzmann et al.

问题：现有视觉相似性度量无法捕捉人类判断中依赖于上下文的多样性（如形状与颜色的区别），缺乏根据特定语义方面进行条件化的机制。
方法：构建大规模三元组图像数据集，标注多种自由形式的语义相似性方面；基于该数据微调视觉语言模型（VLM），提出文本提示图像感知相似性度量（TPIPS），根据文本提示输出不同维度的相似性分数。
结论：TPIPS与人类感知一致性更高，且能泛化到训练分布之外；在文本引导检索、组合搜索和生成模型细粒度评估中展现出新能力。
创新点：首次提出文本提示驱动的多语义视觉相似性度量，通过大规模人类标注数据微调VLM实现条件化相似性判断。
意义：为多模态大模型在细粒度视觉理解、可控检索和生成评估中提供了关键工具，推动感知度量向上下文感知方向发展。

Patch Policy: Efficient Embodied Control via Dense Visual Representations

Gaoyue Zhou, Zichen Jeff Cui, Ada Langford et al.

问题：现有机器人策略要么将观测压缩为单个全局token，损失细粒度空间细节，要么依赖从头训练的视觉骨干，无法利用大规模视觉预训练的优势，而基于密集patch特征的大型视觉-语言-动作模型（VLA）又过于笨重和缓慢。
方法：提出Patch Policy，一种最小架构扩展，通过块因果注意力掩码（block-causal attention mask）使基于Transformer的策略直接消耗预训练的密集patch token，同时保留时间因果性并整合其他状态信息，无需完整VLM的计算开销。
结论：在四个模拟和三个真实世界环境套件中，相比使用最先进全局池化表示的策略，该方法实现了40%的相对改进；相比微调的OpenVLA-OFT，在仅使用约0.7%参数的情况下提升了18%。
创新点：通过轻量级的块因果注意力机制，首次高效地将预训练密集视觉特征直接集成到机器人策略中，避免了全局池化的信息损失和VLA的高计算成本。
意义：为机器人社区提供了一条高效利用视觉表示学习进展的实用管道，在不牺牲训练效率和推理速度的前提下，提升了高频反应式控制性能。

FM-VLA: Force-based Memory for Vision-Language-Action Models in Contact-Rich Manipulation

Ruicheng Li, Qixiu Li, Ruichun Ma et al.

问题：现有记忆增强的视觉-语言-动作模型（VLA）在处理接触丰富的非马尔可夫任务时，基于视觉的记忆方法计算成本高且难以区分视觉模糊的时间事件（如多次小幅度按压按钮）。
方法：提出FM-VLA模型，通过变分自编码器（VAE）将力历史编码为紧凑的力记忆令牌，并与短状态历史一起作为额外条件令牌注入动作专家模块，使VLA能够利用累积的接触事件历史来指导操作。

创新点：首次将基于力的记忆机制引入VLA模型，通过力时间序列的压缩表示解决视觉模糊下的时间上下文推理问题。
意义：为接触丰富的机器人操作提供了高效的非马尔可夫推理方案，推动了VLA模型在细粒度物理交互任务中的应用。

FlowMimic: Mask-free Visual Editing and Generation with Pixel-pair Warped Flow Field for Online Video Editing Data Generation and Modality Mimicry

Dingyun Zhang, Lixue Gong, Wei Liu

问题：现有视频编辑数据收集依赖人工标注、I2V模型合成和VLM过滤，耗时且任务扩展性有限，导致视频编辑任务多样性远低于图像编辑。
方法：提出像素对时间扭曲流场（Pixel-pair Temporal Warped Flow Field），直接从图像编辑样本实时生成对应视频编辑样本；设计模态模仿生成损失和编辑损失，对齐图像与视频模态的能力；引入感知相关任务（如指代分割）及编辑区域感知的潜在层和注意力层损失，使模型内化编辑区域定位能力。

创新点：首次实现无需掩码标注的在线视频编辑数据生成，并通过模态模仿损失统一图像与视频的生成和编辑能力。
意义：为多模态大模型在视频编辑领域的无监督数据生成和统一建模提供了新范式，降低了数据收集成本并提升了任务扩展性。
