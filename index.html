FlashRT: Agent Harness for Guiding Agents to Deploy Real-Time Multimodal Applications

Krish Agarwal, Zhuoming Chen, Yanyuan Qin et al.

问题：实时多模态应用（如语音代理、交互视频生成）的异构模型流水线部署需要应用特定的决策（放置、流式、模型内并行），现有系统无法高效实现自动化优化。
方法：提出FlashRT，一种agent harness，通过新的chain-of-program范式指导编码代理进行多轮变换：将参考实现转化为中间表示（IR）以捕获数据依赖和持久状态范围，通过顺序解释器验证IR，进行静态分析识别候选变换，然后在测量约束的优化循环中迭代实现、验证和基准测试每个候选，生成适应不同硬件预算的高效部署。
结论：在NVIDIA B200 GPU上实现高达~70倍延迟降低和2.8倍吞吐量提升；在AMD MI355X GPU上延迟降低相当，吞吐量提升达3.6倍；对于Qwen3-Omni文本到音频推理，与专家vLLM-Omni实现相比，响应延迟降低65%。
创新点：首次将agent驱动的自动化优化（chain-of-program）应用于实时多模态应用的端到端部署，实现跨平台可扩展的高效部署，无需手动调优。
意义：为世界模型、多模态大模型等实时AI应用的系统部署提供了自动、高效的agent范式，降低了异构硬件上的优化门槛。

SAGE: Subgoal-Conditioned Action Generation for Latent World Model Planning

Letian Cheng, Qi Zhang, Yisen Wang

问题：潜世界模型在长程规划中因固定候选预算下搜索空间指数增长而性能受限。
方法：提出SAGE，使用目标条件生成器预测可到达的潜在子目标作为先验，生成候选动作序列；通过不同时长的子目标平衡细粒度局部控制与长程进展。
结论：在PushT和OGBench Cube上，长程规划成功率大幅提升（如PushT从12.7%到64.7%，OGBench从26.7%到67.3%）。
创新点：将潜在子目标分解与先验条件动作生成结合，替代随机初始化规划。
意义：提升了潜世界模型在长程规划任务中的实用性和性能。

The Many Senses of Visual Similarity: A Text-Prompted Image Perceptual Metric

Sheng-Yu Wang, Yotam Nitzan, Aaron Hertzmann et al.

问题：现有视觉相似性度量无法根据特定语义方面（如形状、颜色）进行条件化判断，将多维度相似性简化为单一标量。
方法：构建一个大规模三元组图像数据集，每个三元组针对多个自由形式的语义方面进行人类相似性标注；基于该数据微调视觉语言模型（VLM），得到文本提示图像感知相似性（TPIPS）度量，可根据文本提示捕捉不同语义维度的相似性。
结论：TPIPS与人类感知一致性更高，能可靠泛化到训练分布之外，并在文本引导检索、组合搜索和生成模型细粒度评估中展现新能力。
创新点：提出一种文本提示驱动的多语义视觉相似性度量，通过大规模人类标注数据和VLM微调实现上下文相关的感知相似性判断。
意义：为多模态大模型在细粒度视觉理解、可解释相似性及可控图像检索任务提供新范式，推动感知度量从单标量向语义条件化发展。

Patch Policy: Efficient Embodied Control via Dense Visual Representations

Gaoyue Zhou, Zichen Jeff Cui, Ada Langford et al.

问题：现有机器人策略要么将观测压缩为单个全局token，要么从头训练视觉骨干，无法有效利用预训练的密集视觉特征，而大视觉语言模型虽能处理密集patch特征但计算开销巨大。本文旨在解决如何在不引入大量参数和推理延迟的前提下，让基于Transformer的策略直接利用预训练的密集patch特征。
方法：提出Patch Policy，通过一种块因果注意力掩码（block-causal attention mask）扩展标准策略架构，使模型能在保持时间因果性的同时，对每个观测的多个patch token以及其他状态信息进行注意力计算，从而直接使用预训练的ViT密集patch特征，无需完整VLM骨干。
结论：在4个模拟和3个真实环境套件中，相比使用全局池化表示的策略，Patch Policy实现了40%的相对性能提升；并且以仅0.7%的参数超越微调后的OpenVLA-OFT 18%的绝对性能。
创新点：设计了一种块因果注意力掩码，使得轻量级策略能高效消耗预训练的密集patch token，避免了全局池化的信息损失和完全微调VLM的高昂成本。
意义：为机器人社区提供了一种即插即用的轻量级管道，能够持续利用视觉表示学习的进步，同时保持高频反应式控制所需的训练和推理效率。

Simple Domain Generalization for Strong Pixel-Level Image Tampering Detection in Modern VLMs

Yi Tang, Xinyi Shang, Jiacheng Cui et al.

问题：现代视觉语言模型（VLM）生成的图像篡改难以检测，尤其在跨模型和分布外场景下，现有方法鲁棒性不足。
方法：提出一个域泛化训练框架，包含平衡小批量采样策略（避免偏向伪造伪影或干净图像先验）和延迟注入策略（先在大型基础数据上训练至稳定收敛，再暴露于少量新兴VLM分布数据）。
结论：在GPT-Images-2.0、Gemini-3.1等OOD VLM上，平均gIoU和cIoU相对提升26.1%和26.8%，显著优于先前最先进方法PIXAR。

意义：为现代VLM图像篡改检测提供了实用且鲁棒的解决方案，有助于提升多模态内容安全领域中的伪造检测可靠性。

FlowMimic: Mask-free Visual Editing and Generation with Pixel-pair Warped Flow Field for Online Video Editing Data Generation and Modality Mimicry

Dingyun Zhang, Lixue Gong, Wei Liu

问题：当前视频编辑数据收集依赖人工掩码标注、I2V模型等复杂流程，任务多样性有限，且现有方法难以在单一模型中统一图像和视频的生成与编辑能力。
方法：提出一种像素对时间扭曲流场，可直接从图像编辑样本实时生成对应的视频编辑样本；设计模态模仿生成损失和编辑损失，通过相互模仿对齐两模态的能力分布；引入感知相关任务和编辑区域感知的潜在层与注意力层损失，使模型内化定位与编辑能力。
结论：实验表明，仅使用该数据训练，模型即可学习多级别视频编辑任务，性能达到或超越依赖外部掩码或MLLM的方法。
创新点：无需掩码标注，通过像素对扭曲流场在线生成视频编辑数据，并通过模态模仿损失统一图像与视频的编辑能力。
意义：为视频编辑数据收集提供高效无掩码方案，推动多模态统一模型的发展，降低对人工标注的依赖。

Sparse Evidence Can Suffice: Agentic Evidence Seeking for Multimodal Video Misinformation Detection

问题：多模态视频虚假信息检测通常作为整体视频理解任务处理，但真实世界虚假信息往往具有稀疏和组合的证据结构，导致冗余和误导性决策。
方法：提出SIEVE框架，解耦证据获取与验证：一个证据寻求智能体主动探索多模态证据并构建紧凑证据包，再由验证者判断真实性；智能体通过监督式证据寻求轨迹和证据感知的强化学习目标训练。

创新点：将证据获取与验证解耦，通过智能体主动寻找稀疏但关键的证据线索，而非处理整个视频，提高了效率和透明度。
意义：为多模态虚假信息检测提供了一种更高效、可解释的范式，推动了世界模型和Agent在可信AI中的应用。

Closing the Loop in Humanoid VLA: Persistent 3D Object Tokens for Verifiable Loco-Manipulation

方法：提出持久对象标记化（POT），从RGB-D观测中维护角色索引的3D对象记录并转换为对象标记，用于全身动作专家；同时利用同一对象记录执行几何谓词检查，形成可动作且可验证的闭环执行系统（POT-VLA）。

创新点：首次提出将持久对象中心化状态同时用于动作条件生成和可验证性检查的闭环人形VLA框架。
意义：为可验证的人形机器人VLA执行提供了一种新的有效抽象，推动了具身智能中长程操作的可信性与鲁棒性。

ST-Veto: Spatio-Temporal Token Veto for Diffusion MLLMs via Taylor Prediction and Visual Grounding

Keuntae Kim, Beomseok Lee, Hyunwoo Kim et al.

问题：扩散多模态大语言模型（dMLLMs）的推理能力尚未充分探索，现有方法存在自回归生成的高成本、错误累积和缺乏自纠正等问题。

方法：提出RoboHarness框架，利用多模态执行记忆和在线证据表征策略能力边界，实现能力感知的任务分解与路由；通过Memory Bridge检索下一策略的执行轨迹，估计其分布内状态区域，并引导机器人进入该区域，实现稳定策略交接。

创新点：将异构策略封装为可重用智能体技能，并通过记忆驱动的能力边界感知与分布匹配实现无联合重训练的稳定编排。
意义：为异构机器人策略的统一编排提供了新的通用范式，有望提升机器人系统在复杂长时域任务中的灵活性和鲁棒性。

Patch Policy: Efficient Embodied Control via Dense Visual Representations

Gaoyue Zhou, Zichen Jeff Cui, Ada Langford et al.

问题：现有机器人策略要么将观测压缩为单个全局token，要么从头训练视觉骨干，无法有效利用预训练的密集视觉特征，而大视觉语言模型虽能处理密集patch特征但计算开销巨大。本文旨在解决如何在不引入大量参数和推理延迟的前提下，让基于Transformer的策略直接利用预训练的密集patch特征。
方法：提出Patch Policy，通过一种块因果注意力掩码（block-causal attention mask）扩展标准策略架构，使模型能在保持时间因果性的同时，对每个观测的多个patch token以及其他状态信息进行注意力计算，从而直接使用预训练的ViT密集patch特征，无需完整VLM骨干。
结论：在4个模拟和3个真实环境套件中，相比使用全局池化表示的策略，Patch Policy实现了40%的相对性能提升；并且以仅0.7%的参数超越微调后的OpenVLA-OFT 18%的绝对性能。
创新点：设计了一种块因果注意力掩码，使得轻量级策略能高效消耗预训练的密集patch token，避免了全局池化的信息损失和完全微调VLM的高昂成本。
意义：为机器人社区提供了一种即插即用的轻量级管道，能够持续利用视觉表示学习的进步，同时保持高频反应式控制所需的训练和推理效率。
